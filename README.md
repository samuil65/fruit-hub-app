# Fruit Hub 🍎

Fruit Hub is a modern Flutter e-commerce application for browsing and ordering fresh fruits with a clean, responsive, and scalable architecture.

## 🚧 Project Status

This project is currently under active development.

New features, UI improvements, and architecture enhancements are continuously being added.

## ✨ Planned Features

- Browse fresh fruits
- Product details page
- Add to cart
- Favorites system
- Search functionality
- User authentication
- Order management
- Responsive UI
- Dark mode support

---

## 🏗 Architecture

The project follows:

- Clean Architecture
- Feature First Structure
- Repository Pattern
- Bloc / Cubit State Management
